Getting Started

Pre-Requisites

  • nodeJS
  • Java Development Kit
  • MySQL Community Server
  • MySQL Workbench

The following software versions used to build this web application include:

  • nodeJS v16.14.2
  • Java 17
  • MySQL Workbench 8.0.29
  • MySQL Community Server 8.0.29
  • Spring Boot 2.6.7
  • Thymeleaf
      • Some kind of text/code editor of your choice. Example:
    • Visual Studio Code
    • IntelliJ IDEA (Community Edition)

Setting Up The Starter Code

The following starter code was generated using this tool called Spring Initializr. This can be found at start.spring.io and the setting in the picture were used to create the starter code.

Initial File Tree

After generating the starter code from the Spring Initializr and extracting the contents, you can open the folding using a text editor or IDE. The picture above shows initial directory tree of this project.
